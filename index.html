SatuSehat FHIR Bridging Module

Undergraduate thesis — a middleware that integrates hospital visit and diagnosis data from a legacy SIMRS with SatuSehat, Indonesia's national health platform. Transforms relational records into FHIR R4 resources (Patient, Encounter, Condition, Observation) with OAuth 2.0 token management. All 12 test scenarios passed against the SatuSehat sandbox, at under 600 ms per resource.

Java Spring Boot 3 REST API HL7 FHIR R4 OAuth 2.0
